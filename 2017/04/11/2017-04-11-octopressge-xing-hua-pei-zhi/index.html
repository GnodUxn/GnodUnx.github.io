Octopress个性化配置

2017-04-11
HTML
octopress, 个性化配置

上篇文章主要讲了使用Octopress搭建个人博客，关于博客的个性化配置将在这篇文章中细述。

###自定义域名
创建了个人博客后，很多小伙伴都迫不及待的想要注册自己的域名了吧。阿里云、腾讯云注册域名都很方便，注册方式这里不再赘述，还没有的小伙伴赶紧去注册自己的域名吧。

这里参照了网上的两种方案。

####方案一

  • 在终端输入：

cd octopress/source # 进入source目录
touch CNAME # 创建一个CNAME文件
vim CNAME # 使用vim编辑器打开,在里面添加你的域名信息（这里不加http://）

cd .. # 返回到octopress根目录
rake generate # 生成静态网页
git add . # 保存到缓存区
git commit -m “log” # 提交代码
git push origin source # 将source分支同步到GitHub服务器
rake deploy # 将master分支同步到GitHub服务器

rake new_page[‘about’] # 新建about界面
vim source/_includes/custom/navigation.html # 编辑navigation.html

vim plugins/category_list_tag.rb ＃ 编辑category_list_tag.rb文件，若没有则新建

module Jekyll
class CategoryListTag < Liquid::Tag
def render(context)
html = “”
categories = context.registers[:site].categories.keys
categories.sort.each do |category|
posts_in_category = context.registers[:site].categories[category].size
category_dir = context.registers[:site].config[‘category_dir’]
category_url = File.join(categorydir, category.gsub(/|\P{Word}/, ‘-‘).gsub(/-{2,}/, ‘-‘).downcase)
html << “

  • #{category} (#{posts_in_category})
    • \n”
    end
    html
    end
    end
    end

    Liquid::Template.register_tag(‘category_list’, Jekyll::CategoryListTag)

    vim source/_includes/asides/category_list.html ＃ 编辑category_list.html，若没有则新建


    文章分类



      {% category_list %}


    vim _config.yml

    excerpt_link: “阅读全部→”

    cd octopress
    git clone https://github.com/bhrigu123/abacus .themes/abacus
    rake install[‘abacus’]
    rake generate
    rake preview # 预览主题

    git add .
    git commit -m ‘Change blog theme.’
    git push origin source
    rake deploy
    ```

    • 其他：在source/images中替换avatar.jpg来自定义头像；在source/_includes/custom/navigation_links.html中编辑标签来自定义导航栏(与上面编辑source/_includes/custom/navigation.html类似)。切记完成操作后执行rake generate命令。
